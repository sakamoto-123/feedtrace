# 宝宝日记

## 一、产品定位

一款专注于新生儿（0-3 岁）日常核心场景记录的工具类 APP，以「简洁、高效、精准」为核心，帮助家长快速记录宝宝喂养、护理、睡眠等关键日常，支持多语言适配、个性化视觉设置，满足不同家庭的使用习惯。

## 二、产品核心目标

1. 让家长快速完成宝宝日常场景记录，无需复杂操作；
2. 清晰呈现记录数据，方便家长回顾、分析宝宝作息规律；
3. 支持个性化视觉设置，提升长期使用的舒适度；
4. 适配不同语言环境，满足跨国 / 多语言家庭需求。

## 三、产品要求（细化规则）

### 1. 国际化支持

- 支持主流语言：支持跟随系统、中文、繁体中文、英文、日文、韩文、法语、德语、葡萄牙语、西班牙语；
- 文案适配规则：所有界面文案、按钮文字、提示语、字段名称均支持根据用户选择的语言自动切换，无翻译遗漏；
- 语言切换路径：设置页 → 语言选择 → 即时生效（无需重启 APP）；
- 补充说明：日期、时间格式支持跟随语言 / 地区自动适配（如中文显示「2024-05-20 14:30」，英文显示「May 20, 2024 2:30 PM」）。

### 2. 深色模式

- 触发方式：① 手动切换（设置页 → 深色模式开关）；② 自动跟随系统（检测手机系统深色模式状态，同步切换）；
- 记忆功能：切换后保存用户偏好，下次打开 APP 保持上次设置。

### 3. 自定义主题色

- 可调整范围：仅支持修改 APP 主色调（按钮、标题栏、选中状态颜色，背景色）；
- 颜色选择：① 预设配色库（10 + 主流安全色，如粉色、蓝色、绿色等，适配宝宝主题）；② 自定义取色（支持 RGB/HEX 输入，或从图片取色，需提示「避免过亮 / 过暗颜色影响显示」）；
- 生效范围：所有界面统一应用自定义主色调，即时预览效果，确认后保存。

# 产品大致描述

## 添加宝宝进入到 APP 之前

用户需要先填写宝宝的基本信息。 包括宝宝的照片、名称、出生日期、性别、体重、身高、头围。

## 添加宝宝进入到 APP 之后

用户添加一个宝宝进入到 APP 之后，APP 底部 Tab 栏有 4 个 tab：首页、 记录、统计、设置

### 首页 TAB

#### 模块 1：宝宝的基本信息模块（常驻顶部）

- 左边是宝宝的头像，头像右侧上方是宝宝的名称、下方是宝宝的年龄
  - 年龄显示方式： xxx 天，xx 月 xx 天，xx 岁 xx 月
- 右侧显示宝宝的体重、身高、头围

#### 模块 2：展示今天宝宝的记录信息统计

- 分为两部分
  - 今天的喂养信息（喝了多少的母乳、多少的奶粉、多少副食、多少补剂）
  - 活动信息：大便了几次、小便了几次，睡觉时常和次数

#### 模块 3：不会常驻的进行中区域

- 有些持续性需要及时的活动卡片显示在这里，如
  - 开始亲喂的时候，会在这里显示亲喂进行中的卡片，上面有具体的亲喂开始时间，亲喂的是左侧还是右侧或者双侧的乳房，亲喂持续时间的计时器，以及一个结束的按钮。
  - 宝宝开始睡觉的时间会在这里显示睡眠进行中的卡片，上面有具体的睡眠开始时间，夜晚还是白天，睡眠持续时间的计时器，以及一个结束的按钮。

#### 模块 4：快速操作区域

- 用户可以点击这个区域的某一项快捷操作，进入到创建记录的页面。
- 用户可以编辑常用的操作进入到快捷操作区域（最多可以选择 8 个）

#### 模块 5：所有操作的区域

- 列举记录类型表中的所有操作

### 记录 TAB

这个页面展示的宝宝所有记录的列表， 由近到远按记录的 startTimestamp 排列。有如下要求：

- 所有的记录的 list 是按天排布的；
- 每个记录列表项的信息如下：
  - 左侧：上方是该记录的 icon；
  - 中侧：
    - 第一排是该记录的名称 + · 记录的内容（记录的内容不同类型不一样，具体见记录类型表）；
    - 第二排是 记录的开始时间：xx:xx xx 分钟前/xx 秒前/xx 小时前/xx 天前/xx 月前（具体时间，精确到小时和分钟） + 该记录的备注（如果 remark 字段有值的话，最多显示两行，超出显示省略号）；
  - 右侧：该记录的图片列表（如果存在的话），图片列表是 36 \* 36 的大小，一张压在另一张的上方，堆叠显示的；
  - 每个记录列表项，可以左滑，有删除 + 编辑的按钮， 点击编辑按钮进入到记录的编辑页面；
  - 点击列表项进入到该记录的详情页面。

#### 创建记录和编辑记录的页面

这个页面主要用于创建记录的。创建记录的页面内容包括如下：

- 时间选择模块（可以选择日期+时间，时间精确到小时和分钟）
  - 开始时间：所有记录类型都需要；
  - 结束时间：一些需要持续进行的记录需要这个字段，比如亲喂、睡觉。
- 信息模块
  - 输入名称（name 字段， 只有就医、用药、补剂、疫苗记录需要）；
  - 喂养侧（breatType 字段，记录亲喂类型的时候需要）；
  - 记录用量（数字输入框 + 单位）：value 和 unit 字段（记录类型表里面有 value + unit 的类型需要）；
  - 记录睡觉的时候需要选择白天还是黑夜（dayOrNight）；
  - 记录辅食的时候需要选择辅食接受度（acceptance 字段）；
  - 记录纸尿裤的时候需要选择排泄物的类型（excrementStatus 字段）。
- 补充信息（所有记录模块都有）
  - 备注输入框（remark 字段）；
  - 添加照片（可以调用相机 + 相册， photos 字段）。
- 操作模块
  - 新建记录的时候：最下方是一个保存的按钮；
  - 编辑记录的时候：两个按钮，左边是删除、右边是保存按钮。

#### 查看记录详情的页面

- 内容同创建/编辑记录的页面，只是内容都是以文本或者图片的形式展示的；
- 右上角有一个编辑的按钮，点击编辑进入到记录的编辑页面。

### 统计 TAB

统计页面上方有四个选择，分别是喂养趋势、睡眠趋势、成长统计和每日活动。

#### 喂养趋势

- 可以选择看最近 7 天、30 天、90 天
- 第一张图： 横坐标是日期，纵坐标是奶量， 区分奶粉还是母乳；
- 第二张图：横坐标是日期，纵坐标是次数， 区分奶粉还是母乳。

#### 睡眠趋势

- 可以选择看最近 7 天、30 天、90 天
- 第一张图： 横坐标是日期，纵坐标是睡眠时间；
- 第二张图：横坐标是日期，纵坐标是睡眠次数。

#### 成长统计

- 可以选择看最近 7 天、30 天、90 天
- 统计维度：身高、体重、头围、BMI。

#### 每日活动

- 横坐标是（日期+周几）显示最近 7 天的数据；
- 纵坐标是一天的 24 小时；
- 每个格子里显示的选择的当天，某个时辰宝宝进行的活动。

  - 模式 🌗
